# v-ops
vantop devops运维平台

开发环境： Linux Centos7.6 . Python3.8  Django3.1  MySQL5.7 Redis 4.0

Python libs： ./requirements.txt


该项目可以使用LDAP账号登录，结合Jenkins做服务管理和自动化发布。
